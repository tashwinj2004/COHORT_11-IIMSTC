# The Food Waste Reduction System 
It is a machine learning-based project designed to predict the amount of food waste generated by retail outlets by analyzing patterns in inventory movement, product lifespan, and sales behavior. 
The system applies data preprocessing and feature engineering techniques to capture key factors influencing spoilage and unsold stock, followed by the implementation of regression models to accurately estimate waste quantities. Model performance is evaluated using regression metrics to ensure reliability and effectiveness. 
The final project is organized within the FOODWASTE_REDUCTION_SYSTEM folder, containing all modules related to preprocessed datasets, model development and evaluation. 
The overall objective is to support data-driven retail decisions, reduce food wastage, improve operational efficiency, and promote sustainable consumption practices aligned with SDG 12 – Responsible Consumption and Production.
